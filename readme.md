# Projet gestion de don et besoin pour la BNGRC

Examen final web S3 

## Configuration

Modifier le nom de db et mdp dans app/config.php

Le script de database se trouve dans database/script_base.sql

## Lancement 

Pour prendre les dependances php

```bash

composer update

```

Lancer le projet 

```bash

php -S localhost:8000 -t public 

```
